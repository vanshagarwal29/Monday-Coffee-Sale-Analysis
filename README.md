"# Monday-Coffee-Sale-Analysis" 
This project includes SQL queries made in MySQL Workbench for the Monday Coffee dataset.

## About
The goal is to study coffee sales, customers, and city data to find useful insights.

## Files
- **MONDAY_COFFEE_1.SQL** → First 5 questions (Q1–Q5)  
- **MONDAY_COFFEE_2.SQL** → Next 5 questions (Q6–Q10)

## Main Topics
- Total revenue and customers  
- Top cities and products  
- Monthly sales growth  
- Coffee consumers and rent details

## Tool Used
MySQL Workbench
